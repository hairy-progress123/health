# Health Sync AI

Health Sync AI is a comprehensive healthcare platform designed to empower patients by connecting them directly with medical professionals and providing intelligent health assistance.

## Key Features

- **Doctor Video Consultations**: Seamlessly connect with your doctors through high-quality video calls from anywhere.
- **AI Health Assistant**: Ask questions about your health problems, analyze medical reports or images, and get instant, helpful AI-driven insights.
- **Progress Monitoring**: Track, monitor, and visualize your health progress directly through the app.

---

## Getting Started

First, install the dependencies (if you haven't already):
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the app. for more details.
